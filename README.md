# Szakdolgozat – Python forráskód

Ez a repository a szakdolgozatomhoz tartozó Python forráskódokat tartalmazza.  
A kód egy szimulációs és teljesítményattribúciós keretrendszert valósít meg, amely opciós portfóliók, benchmark portfólió, aktívan kezelt portfóliók, Black–Scholes-árazás, görög betűs hozamfelbontás és Brinson-típusú teljesítményattribúciós modellek vizsgálatára szolgál.

## Tartalom

- `main.py` – a teljes szimulációs és attribúciós folyamat fő futtatófájlja
- `benchmark_setup.py` – a benchmark portfólió kezdeti szerkezetének létrehozása
- `piaci_allapotvaltozok.py` – időszak végi piaci állapotváltozók szimulációja
- `Black_Scholes.py` – Black–Scholes opcióárazás és görög betűk számítása
- `hozamok.py` – opció-, szektor- és benchmarkszintű hozamfelbontás
- `aktiv_portfoliok.py` – aktívan kezelt portfóliók generálása
- `Brinson.py` – klasszikus és saját teljesítményattribúciós modellek számítása
- `abrak.py` – a szimulációs eredmények grafikus megjelenítése
- `tablazatok.py` – szakdolgozatba illeszthető Word-táblázatok előállítása
